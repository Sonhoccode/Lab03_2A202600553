Công Cụ Có Sẵn
KEY INSIGHT
"Chatbot đoán — Agent biết."

Khi bạn hỏi "Giá vé rẻ nhất HAN→SGN ngày 12/06 là bao nhiêu?", Chatbot sẽ đưa ra con số nghe có vẻ hợp lý nhưng hoàn toàn ảo. ReAct Agent gọi search_flights(), lấy dữ liệu thực, rồi gọi calculate_total_price() để tính đúng số tiền với thuế.

❌ Chatbot
"Giá vé thường dao động từ 800.000đ đến 2.500.000đ..."
(đoán mò, không có dữ liệu thực)
✅ ReAct Agent
Thought → search_flights(HAN,SGN,2026-06-12)
Obs → [VJ801: 1.500.000đ, VJ201: 1.650.000đ, ...]
Thought → calculate_total_price(VJ801, 1, [])
Obs → total: 1.650.000đ (với 10% thuế)
Final: VJ801 - 1.650.000đ
